## Featured Project

### CourseCrash — Coursera Clone  
Online learning platform (In Progress)

**Tech Stack**  
Node.js · Express · MongoDB · Redis · Next.js

**Key Learnings**
- Backend-first architecture scales better over time
- Course and lesson management is rarely “just CRUD”
- Redis improves performance when used intentionally
- Clear separation of frontend and backend prevents future confusion

This project exists because building something real exposes problems no tutorial prepares you for.
